Fake Store Api

Fake store rest API for your e-commerce or shopping website prototype

Get tired of Lorem ipsum data?! Didn't you find any free e-commerce API?!

fakeStoreApi is a free online REST API that you can use whenever you need Pseudo-real data for your e-commerce or shopping website without running any server-side code. It's awesome for teaching purposes, sample codes, tests, etc.

Example Code

fetch('https://fakestoreapi.com/products/1') .then(res=>res.json()) .then(json=>console.log(json))

Resources

There are 4 main resources needs in shopping prototype.

  • Products

    20 Products

  • Cart

    20 Cart Items

  • Users

    10 Users

  • Login Token

Routes

All HTTPS methods are supported.

  • GET

    /products

  • GET

    /products/1

  • GET

    /product/categories

  • GET

    /product/categories/electornics

  • GET

    /product/userid?=1

  • POST

    /product

  • PUT

    /product/1

  • PATCH

    /product/1

  • DELETE

    /product/1
